# Icone per le Categorie della Tier List

Inserisci in questa cartella le icone (formato `.png` o `.svg`) da utilizzare per ogni categoria presente nella Tier List.

Assicurati che i nomi dei file corrispondano **esattamente** ai nomi delle categorie nel database (tutto minuscolo), ad esempio:
- `crystal.png` (o `crystal.svg`)
- `sword.png`
- `axe.png`
- `pot.png`
- `uhc.png`
- `smp.png`

Il sito proverà a caricare il file `.png` o `.svg` con il nome della categoria selezionata.
